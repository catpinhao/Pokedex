# Pokédex
This is just a small side project to showcase all different Pokémon from all 8 generations, using [PokeAPI](https://pokeapi.co/). Accessible at https://lucid-ramanujan-a44256.netlify.app/

## Requirements
- `yarn` or `npm`

## Setup and Run
- clone repository
- `cd pokedex`
- `yarn start` or `npm start`
